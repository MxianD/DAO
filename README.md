# DAO

## 使用技术
  ### 前端 vue3
  ### 链端 solidity
  ## 后端  java
## 使用框架
  ### Truffle
  ### Vue-cli
  ### springboot
## 使用说明
  本案例运行在本地搭建的区块链上
  ### 1.下载MetaMask
  ### 2.下载Ganache
  ### 3.运行Ganache(Quick start)
  ### 4.进入根目录运行truffle migrate
  ### 5.运行npm run dev
  ### 6.MetaMask根据Ganache连接本地区块链网络
  ### 7.根据Ganache导入私钥创建钱包账户
  ### 8.开始使用
